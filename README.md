# MOVIEMAX

#### Step 1: Install Node.js dependencies:
     npm install        
#### Step 2: Set your api key environment variable(.env). Get an api key from  [TheMovieDb](https://www.themoviedb.org/)
     REACT_APP_API_KEY=your_api_key
#### Step 3:
     npm  start
